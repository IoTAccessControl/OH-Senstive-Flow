1 我们如何收集和使用您的个人信息

在“当用户打开应用首页时，系统会自动记录页面加载状态以便后续问题排查”场景中，我们会从日志标签 (logTag)、日志域 ID (domainId)、组件生命周期状态收集日志标签 (logTag)、日志域 ID (domainId)和组件生命周期状态，用于开发调试与运行状态监控，不上传至云端。

在“用户进入个人信息页面查看脱敏手机号及执行退出登录操作时”场景中，我们会从本地持久化存储收集匿名化手机号、组件日志标签、日志域 ID和登出操作状态，用于展示用户个人信息、记录应用运行状态与用户操作行为以便调试。此外，在“用户在个人信息页面或快速登录页面点击隐私协议链接时”场景中，我们会从系统配置、网络状态接口、应用资源文件收集系统语言环境、网络连接状态、隐私协议文本内容和隐私协议 URL 参数，用于根据用户设备环境加载对应语言的隐私政策页面，确保用户知情权。此外，在“用户按下返回键尝试退出应用或页面时”场景中，我们会从用户交互事件、系统异常对象收集用户返回操作事件、应用终止错误码和应用终止错误消息，用于响应用户退出请求，记录应用终止过程中的异常情况用于故障排查。

在“用户点击隐私协议链接时，应用记录操作日志并检查网络状态以加载对应语言的协议页面”场景中，我们会从用户交互事件、系统环境信息收集用户操作事件（点击协议链接）、日志域标识符 (domainId)、日志标签 (logTag)、网络连接状态、系统语言环境、错误码 (error.code)、错误消息 (error.message)和隐私协议文本内容，用于记录用户交互行为以便调试，获取设备环境信息以适配多语言协议展示，监控网络状态以确保协议页面可加载。

在“用户点击登录按钮触发快速登录流程时，应用通过华为账号服务获取用户的匿名手机号、UnionID 和 OpenID，用于后续业务验证”场景中，我们会从系统账号服务 (HuaweiIDProvider)收集匿名手机号 (quickLoginAnonymousPhone)、UnionID和OpenID，用于实现用户身份的快速验证与无感登录。此外，在“用户在登录准备页查看或点击隐私协议链接时，应用记录相关操作日志及错误信息”场景中，我们会从用户交互事件、系统配置收集系统语言环境、网络连接状态和异常错误码与消息，用于调试追踪、错误诊断及多语言协议适配。

在“加载并展示用户协议与隐私政策网页内容时，收集设备屏幕尺寸以适配布局，并在页面加载过程中记录进度及潜在错误信息”场景中，我们会从系统显示服务 (display.getDefaultDisplaySync)、WebView 组件内部状态收集设备屏幕物理宽度、Web 页面加载进度、组件标识符 (domainId, logTag)和错误码与错误消息，用于实现响应式 UI 布局适配，监控资源加载状态，并在发生异常时生成诊断日志以便问题排查。

在“用户进入快速登录页面时，系统自动记录页面加载事件及匿名手机号参数，用于后续业务逻辑判断”场景中，我们会从应用内部状态收集匿名手机号 (anonymousPhone)、日志标签 (logTag)和域 ID (domainId)，用于支持快速登录功能的初始化与调试信息记录。此外，在“用户点击返回按钮时，系统记录当前页面栈深度信息以执行回退操作”场景中，我们会从应用内部状态收集页面栈大小 (pageInfos.size)、日志域 ID (domainId)和日志标签 (logTag)，用于实现页面导航回退逻辑及相应的操作审计。此外，在“用户点击隐私协议链接时，系统获取设备网络状态和系统语言环境，构建目标页面 URL 并跳转”场景中，我们会从设备系统配置、网络连接状态收集系统语言设置 (systemLanguage)、隐私协议文本内容 (Privacy Policy Text)和网络连接状态，用于根据用户设备语言环境展示对应的隐私政策内容。

2 设备权限调用

在“Checks whether the default data network is activated.”场景中，我们会调用ohos.permission.GET_NETWORK_INFO权限，用于使用相关系统能力（由 SDK API 权限映射确定）。若您拒绝授权，拒绝授权可能导致对应功能无法正常使用。此外，在“用户点击隐私协议链接准备加载网页时”场景中，我们会调用ohos.permission.INTERNET权限，用于检查网络连接状态并加载远程隐私政策页面。若您拒绝授权，无法检测网络状态，可能导致在无网环境下尝试加载失败或无法打开隐私政策页面。

在“Checks whether the default data network is activated.”场景中，我们会调用ohos.permission.GET_NETWORK_INFO权限，用于使用相关系统能力（由 SDK API 权限映射确定）。若您拒绝授权，拒绝授权可能导致对应功能无法正常使用。此外，在“用户点击隐私协议链接后，应用尝试加载远程协议页面”场景中，我们会调用ohos.permission.INTERNET权限，用于隐含依赖，用于支持 WebView 加载远程隐私协议内容及网络连通性检查。若您拒绝授权，若拒绝该权限，用户将无法查看在线版本的隐私协议内容。

在“Checks whether the default data network is activated.”场景中，我们会调用ohos.permission.GET_NETWORK_INFO权限，用于使用相关系统能力（由 SDK API 权限映射确定）。若您拒绝授权，拒绝授权可能导致对应功能无法正常使用。此外，在“应用发起华为账号授权请求以获取用户标识符，以及加载远程隐私协议网页内容”场景中，我们会调用ohos.permission.INTERNET权限，用于建立与服务器的网络通信，完成身份验证握手及动态资源加载。若您拒绝授权，若拒绝此权限，应用将无法连接华为账号服务获取用户信息，导致快速登录功能完全不可用；同时无法加载在线的隐私协议文本。

在“Checks whether the default data network is activated.”场景中，我们会调用ohos.permission.GET_NETWORK_INFO权限，用于使用相关系统能力（由 SDK API 权限映射确定）。若您拒绝授权，拒绝授权可能导致对应功能无法正常使用。此外，在“检查网络连接状态及加载远程隐私政策页面”场景中，我们会调用ohos.permission.INTERNET权限，用于调用 hasDefaultNetSync() 检测网络连通性，以及潜在地加载远程资源。若您拒绝授权，若拒绝此权限，应用可能无法验证网络状态，导致无法正确跳转或显示错误提示。
