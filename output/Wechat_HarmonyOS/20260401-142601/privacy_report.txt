1 我们如何收集和使用您的个人信息

当您在使用通讯录功能时，为支持您通过点击侧边字母索引快速定位联系人分组，我们会处理您的用户交互索引位置、通讯录分组标题及列表滚动目标索引。相关数据仅在本地内存中临时处理用于实现界面跳转与操作反馈，不会进行本地持久化存储，也不会上传至应用服务端。

为了向您展示通讯录列表并支持您点击联系人发起聊天，我们会在应用本地内存中临时存储和展示您的联系人姓名和联系人头像。当您点击联系人时，相关数据仅在应用内部页面间传递，不会上传至应用服务端。

为了向您展示当前设备的硬件配置、系统环境及网络连接状态信息，当您访问硬件信息页时，我们会收集您的设备市场名称、操作系统全称、SDK API 版本、设备类型、系统运行时间、屏幕分辨率、屏幕刷新率、屏幕像素密度、传感器 ID 列表、默认网络连接状态、网络承载类型、IP 地址及子网掩码。相关数据仅在本地处理，用于界面渲染，不会上传至应用服务端。

我们在您浏览微信首页时，会在本地内存中临时读取并处理“聊天列表数据”和“用户昵称”，用于向您展示聊天会话列表；当您点击特定会话时，相关数据会作为页面跳转参数传递至聊天详情页面，以支持您进入具体的聊天界面。上述数据仅在设备本地处理，不会上传至应用服务端。

在您尝试更换头像操作失败时，我们可能会接收系统返回的错误信息，并仅在本地通过界面提示向您展示该错误信息，以告知操作失败的具体原因。相关数据仅在本地处理，不会被存储或上传至应用服务端。

在您使用搜索页面并点击“取消”按钮时，为了判断当前页面层级以决定是关闭键盘还是返回上一级页面，我们会读取您的页面栈深度和键盘高度状态。相关数据仅在本地处理，不会上传至应用服务端，且仅在内存中临时存储用于逻辑判断，无持久化存储行为。

2 设备权限调用

在“用户在聊天页面使用语音输入功能”场景中，我们会调用麦克风权限（动态授权），用于录制用户语音以发送语音消息。若您拒绝授权，无法使用语音输入功能，系统会提示用户先授权麦克风权限。

在“用户在硬件信息页查看网络详情时，应用获取具体的网络承载类型、IP 地址及子网掩码”场景中，我们会调用获取网络信息权限（预授权），用于获取默认网络连接对象、网络能力集以及详细的网络连接属性（含 IP 地址）。若您拒绝授权，若拒绝该权限，应用无法获取网络类型、IP 地址、子网掩码等详细网络信息，相关字段将显示为空或不可用。

在“用户在个人中心点击头像以从系统相册选择新图片”场景中，我们会调用读取媒体内容权限（动态授权），用于允许应用访问用户媒体库以读取选中的头像图片路径。若您拒绝授权，用户无法从相册选择图片来更换头像，只能使用默认头像或取消操作。

当前已在应用源码/配置扫描或 SDK API 权限映射中识别到以下权限：相机权限（动态授权）、网络访问权限（预授权）、振动权限（预授权）、写入媒体内容权限（动态授权）；但尚未定位到可回溯的功能点数据流，因此本章节暂不生成具体权限声明。
