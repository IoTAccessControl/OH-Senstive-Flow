1 我们如何收集和使用您的个人信息

在“应用启动时记录日志”场景中，我们会从应用启动事件收集应用启动生命周期状态，用于记录应用启动的生命周期事件。相关数据仅在本地处理。

在“应用退出时”场景中，我们会从应用生命周期状态收集应用销毁事件标记，用于调试与日志记录。相关数据仅在本地处理。

在“应用启动并创建主窗口时”场景中，我们会从系统事件收集应用生命周期状态、窗口创建事件、错误对象、错误代码、错误消息字符串、页面加载状态数据和错误信息详情，用于记录应用启动及页面加载过程中的成功或失败状态，用于调试和故障排查。相关数据仅在本地处理。

在“应用主窗口销毁时”场景中，我们会从应用生命周期状态收集窗口阶段销毁事件和调试日志记录，用于释放 UI 相关资源及记录调试信息。相关数据仅在本地处理。

在“应用切到前台时”场景中，我们会从系统生命周期事件收集应用生命周期状态（前台/后台切换事件）和固定日志标识字符串，用于记录应用运行状态日志。相关数据仅在本地处理。

在“应用切到后台时”场景中，我们会从应用生命周期状态收集应用进入后台事件，用于系统调试与运行状态记录。相关数据仅在本地处理。

在“聊天页面加载时获取用户名称”场景中，我们会从路由跳转参数收集用户名称，用于初始化聊天页面显示内容。相关数据仅在本地处理。此外，在“用户点击语音输入按钮时检查麦克风权限”场景中，我们会从用户交互事件、系统权限状态收集麦克风权限状态和用户交互事件（点击语音按钮），用于确保用户授权后方可使用语音输入功能。相关数据仅在本地处理。

在“展示通讯录列表及点击联系人进入聊天页面”场景中，我们会从本地内存数据收集联系人姓名和联系人头像标识，用于在通讯录界面展示联系人信息并支持用户点击进入聊天。相关数据仅在本地处理。此外，在“通过字母索引快速定位通讯录分组”场景中，我们会从本地内存数据收集通讯录列表索引和字母索引，用于响应用户点击侧边栏字母以滚动至对应联系人分组。相关数据仅在本地处理。

在“页面展示时自动收集并显示设备硬件及网络状态信息”场景中，我们会从系统设备信息模块、系统进程管理模块、系统显示服务、系统传感器服务、系统网络连接服务收集设备市场名称、操作系统全名、SDK API 版本、设备类型、系统运行时间、默认网络标识、屏幕宽度、屏幕高度、屏幕刷新率、屏幕像素密度、支持的传感器 ID 列表、网络承载类型、IP 地址和子网掩码，用于向用户展示当前设备的硬件配置、系统版本及网络状态详情。相关数据仅在本地处理。

在“聊天列表展示与点击进入聊天页面”场景中，我们会从用户昵称、聊天列表数据收集用户昵称和聊天列表数据，用于渲染聊天列表界面并支持用户点击跳转至具体聊天详情页。相关数据仅在本地处理。

在“用户点击头像更换图片”场景中，我们会从相册收集用户头像图片URI和相册选择错误信息，用于更新用户头像显示。相关数据仅在本地处理。此外，在“用户点击操作跳转至个人二维码页面”场景中，我们会从UI交互事件收集UI交互事件（点击操作）和页面路由参数（目标页面路径），用于导航至个人二维码展示页面。相关数据仅在本地处理。

在“用户在搜索页面点击取消按钮或返回键时，应用判断当前路由状态并执行页面返回操作”场景中，我们会从系统路由栈信息、本地内存变量收集路由栈长度、键盘高度状态和页面返回事件，用于控制页面导航行为，确保在特定条件下（如键盘展开或处于首页）正确响应返回操作。相关数据仅在本地处理。

在“搜索页面返回操作与键盘状态管理”场景中，我们会从用户按键输入收集键盘高度状态和按键事件类型，用于判断键盘是否展开以决定返回键行为及显示提示消息。相关数据仅在本地处理。

在“用户输入搜索关键词并查看结果提示”场景中，我们会从用户手动输入收集用户搜索关键词，用于响应用户搜索请求并在界面显示输入内容反馈。相关数据仅在本地处理。

2 设备权限调用

在“用户点击语音输入按钮时”场景中，我们会调用麦克风权限（动态授权），用于获取麦克风权限以支持语音输入功能。若您拒绝授权，若拒绝授权，将提示“请先授权麦克风权限”，无法使用语音输入功能。

在“获取网络连接状态、IP 地址及网络类型等网络相关信息”场景中，我们会调用获取网络信息权限（预授权），用于允许应用检查默认网络是否存在，并获取 IP 地址、子网掩码及网络承载类型（如 WiFi/蜂窝）。若您拒绝授权，无法显示网络类型、IP 地址、子网掩码及广播地址等网络状态信息。

在“用户点击头像更换图片”场景中，我们会调用读取媒体内容权限（动态授权），用于访问相册以获取用户选择的图片。若您拒绝授权，无法从相册选择图片进行头像更换。

当前已在应用源码/配置扫描或 SDK API 权限映射中识别到以下权限：相机权限（动态授权）、网络访问权限（预授权）、振动权限（预授权）、写入媒体内容权限（动态授权）；但尚未定位到可回溯的功能点数据流，因此本章节暂不生成具体权限声明。
