1 我们如何收集和使用您的个人信息

在“请求插屏视频广告时收集设备标识与错误日志”场景中，我们会从应用内部逻辑、系统广告服务收集OAID (Open Anonymous Device Identifier)、用户追踪同意状态、错误代码 (Error Code)和错误消息 (Error Message)，用于向广告服务器请求并展示个性化的插屏视频广告内容，以及诊断广告加载过程中的技术故障。

2 设备权限调用

在“在页面加载初始化阶段申请用户是否同意被跨应用追踪的授权”场景中，我们会调用ohos.permission.APP_TRACKING_CONSENT权限，用于获取用户的明确同意，以便合法合规地收集和使用 OAID 进行广告归因和个性化推荐。若您拒绝授权，如果用户拒绝此权限，应用可能无法获取有效的 OAID，导致无法展示个性化广告或广告加载失败。此外，在“Subscribes to a common event. This API uses an asynchronous …”场景中，我们会调用ohos.permission.RECEIVE_SMS权限，用于使用相关系统能力（由 SDK API 权限映射确定）。若您拒绝授权，拒绝授权可能导致对应功能无法正常使用。
