1 我们如何收集和使用您的个人信息

我们仅在本地处理您的个人信息，包括应用生命周期事件标识、错误码、错误消息及页面加载结果状态，用于记录应用启动过程及页面加载状态以进行运行监控和故障排查。相关数据仅写入系统日志缓冲区，无持久化存储，且不会上传至应用服务端。

我们仅在您点击按钮时，通过调用系统接口异步读取“窗口宽度”信息，用于在页面展示相关数据；该数据仅存储在组件内存状态中，未持久化且未加密，不会上传至应用服务端。

2 设备权限调用

当前已在应用源码/配置扫描或 SDK API 权限映射中识别到以下权限：访问蓝牙权限（预授权）、获取WIFI信息权限（预授权）；但尚未定位到可回溯的功能点数据流，因此本章节暂不生成具体权限声明。
