1 我们如何收集和使用您的个人信息

在“用户点击按钮触发系统信息获取操作，用于在页面展示设备屏幕尺寸、品牌或方向等基础设置”场景中，我们会从设备系统服务收集窗口宽度 (windowWidth)、设备品牌 (brand)和设备方向 (deviceOrientation)，用于在应用界面动态展示当前设备的屏幕参数及硬件标识信息。

2 设备权限调用

当前已在应用源码/配置扫描或 SDK API 权限映射中识别到以下权限：ohos.permission.ACCESS_BLUETOOTH、ohos.permission.GET_WIFI_INFO；但尚未定位到可回溯的功能点数据流，因此本章节暂不生成具体权限声明。
