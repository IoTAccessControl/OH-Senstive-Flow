# HarmonyOS 应用隐私声明报告

## 模块概述
**功能模块：** network_info  
**代码位置：** HardwareIndexPage.ets (第62-64行)

## 1. 主要数据类型
- **网络连接状态信息**：设备当前的网络连接情况（是否有默认网络连接）

## 2. 数据收集和使用目的
- **收集内容**：检测设备是否具有默认网络连接
- **使用目的**：用于条件判断，可能用于控制页面功能或提示用户网络状态
- **API调用**：使用HarmonyOS connection API的`hasDefaultNetSync()`方法

## 3. 隐私风险评估
**低风险**

该模块仅获取设备基础网络连接状态，不涉及具体的网络类型、IP地址、MAC地址等敏感信息，对用户隐私影响较小。

## 4. 隐私保护建议
- ✅ 该功能符合最小化数据收集原则
- ✅ 建议在用户协议中明确说明网络状态检测用途
- ✅ 如非必要，可考虑提供用户选择是否启用网络检测功能
- ✅ 确保网络状态信息仅在本地使用，不上传至服务器
