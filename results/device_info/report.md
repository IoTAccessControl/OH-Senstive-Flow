# HarmonyOS应用隐私声明报告 - device_info模块

## 主要数据类型
- **设备信息**：设备型号名称(marketName)
- **系统信息**：操作系统全名(osFullName) 
- **开发版本**：SDK API版本号(sdkApiVersion)
- **设备类型**：设备分类标识(deviceType)

## 数据收集和使用目的
- **调试与日志记录**：通过`hilog.info`将设备信息输出至应用日志
- **硬件信息展示**：在HardwareIndexPage页面中获取并显示设备硬件相关信息
- **系统兼容性检测**：获取系统版本和API级别用于适配性判断

## 隐私风险评估
**中等风险**

虽然收集的主要是设备硬件和系统信息（非个人身份信息），但完整的设备指纹信息可能被用于设备识别和追踪。

## 隐私保护建议
1. **限制日志输出**：避免在生产环境中输出详细的设备信息日志
2. **最小化原则**：仅收集应用运行必需的设备信息
3. **用户告知**：在隐私政策中明确说明设备信息的收集和使用目的
4. **数据保护**：对存储的日志信息采取适当的加密保护措施
