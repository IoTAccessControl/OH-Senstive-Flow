# HarmonyOS 应用隐私声明报告（功能模块：device_info）

## 1 我们如何收集和使用您的个人信息

为了提供设备信息展示与调试支持，我们可能会在您授权的情况下收集部分设备基本信息。这些信息仅用于本地日志记录和开发调试用途，不会上传或共享给第三方。

### 1.1 设备信息获取与日志输出

为了展示设备型号，我们会读取 `marketName` 属性 [[flow_id:1, step:0]]，并将其通过日志系统输出 [[flow_id:1, step:1]]。

为了展示操作系统全名，我们会读取 `osFullName` 属性 [[flow_id:2, step:0]]，并将其通过日志系统输出 [[flow_id:2, step:1]]。

为了展示 SDK API 版本号，我们会读取 `sdkApiVersion` 属性 [[flow_id:3, step:0]]，并将其通过日志系统输出 [[flow_id:3, step:1]]。

为了展示设备类型，我们会读取 `deviceType` 属性 [[flow_id:4, step:0]]，并将其通过日志系统输出 [[flow_id:4, step:1]]。

**重要提醒**：以上所有数据均仅在本地处理，并通过 HarmonyOS 的 `hilog.info` 接口输出至系统日志，不涉及网络传输或用户隐私泄露风险。

---

## 2 设备权限调用

本模块 **无需申请任何运行时权限**，因为所使用的设备信息接口为 HarmonyOS 提供的系统级 API，属于非敏感信息访问范畴。
